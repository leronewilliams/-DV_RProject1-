In order to plot each data frame, the ggplot2 package was loaded and a different geom, short for geometric objects was chosen as follows: df1 = geom_point(), df2 = geom_path(), df3 = geom_density2d(), df4 = geom_dotplot()

source("../02 Visualizations/ggplot.R", echo = TRUE)
